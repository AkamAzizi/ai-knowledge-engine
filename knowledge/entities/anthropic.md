# Anthropic

## Overview

AI safety company and creator of the Claude family of LLMs. Developed the Model Context Protocol (MCP) as an open standard for connecting AI agents to tools and external services. Publishes engineering and research findings through their blog and documentation.

## Key Contributions (in this wiki)

- Developed [[concepts/model-context-protocol|Model Context Protocol (MCP)]]
- Published evaluation-driven methodology for [[concepts/agent-tool-design|agent tool design]]
- Internal tools (Slack, Asana workspace integrations) served as the primary test bed for the tool design principles described in [[sources/writing-effective-tools-for-ai-agents|Writing Effective Tools for AI Agents]]
- Claude Code used as the agent collaborator for analyzing evals and improving tool implementations
