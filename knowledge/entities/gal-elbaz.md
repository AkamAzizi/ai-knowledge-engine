# Gal Elbaz

## Overview

Co-Founder and CTO at [[entities/oligo-security|Oligo Security]]. Over a decade of expertise in vulnerability research and ethical hacking. Background: security engineer in the IDF's elite intelligence unit; later senior security researcher at Check Point where he helped build the research team.

## Role

Provided expert tips on advanced secure coding practices in the Oligo article, focusing on CI/CD exploit simulation, pre-commit security hooks, security type abstractions, CVE ownership linking, and runtime monitoring.

## Key Contributions

- Advocate for **runtime security monitoring** as a complement to static secure coding
- Proponent of **security type wrapper libraries** (`SafeSQL`, `SafeHTML`, `SafeJWT`)
- Recommends linking **code ownership to CVE alerts** via GitHub CODEOWNERS
