# Oligo Security

## Overview

Security company specializing in runtime application security — monitoring applications in production to detect and block vulnerabilities that slip past static code analysis and secure coding practices.

## Key Contributions

- Provides the **runtime vulnerability management** layer: monitors applications in production for anomalous behavior and blocks exploitation of missed vulnerabilities.
- Publishes the Oligo Security Academy, covering application security topics.
